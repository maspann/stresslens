Tentang StresLens

Sistem pakar untuk memprediksi tingkat beban dan stres mahasiswa S1 berdasarkan kombinasi faktor akademik, fisik, dan aktivitas non-akademik — menggunakan logika fuzzy untuk merepresentasikan ambiguitas dan interaksi non-linear antar-faktor.

Masalah yang dijawab

Mahasiswa cenderung tidak menyadari bahwa beban mereka sudah melampaui kapasitas sampai gejala fisik dan psikis muncul. Kesadaran yang terlambat menyebabkan intervensi (istirahat, restrukturisasi prioritas, konsultasi konselor) baru dilakukan setelah kondisi memburuk. StresLens memberikan peringatan dini berbasis self-input, memungkinkan refleksi sebelum titik kewalahan tercapai.

Mengapa Fuzzy Logic, bukan if-else?

Tiga alasan teknis menjustifikasi pemilihan fuzzy:

  • Variabel linguistik ambigu. "Kurang tidur" atau "tugas banyak" tidak punya batas tegas. Crisp logic memaksa threshold buatan (misal: kurang = <6 jam) yang tidak mengakomodasi variasi personal.
  • Interaksi non-linear. Tugas banyak + tidur cukup ≠ tugas banyak + tidur kurang. Efek kombinasinya tidak proporsional dan butuh inference engine yang merepresentasikan reasoning seperti konselor.
  • Gradasi keparahan. "Sedikit kewalahan" vs "sangat kewalahan" lebih sesuai dengan realitas psikologis ketimbang biner kewalahan/tidak.

Arsitektur Sistem

StresLens menggunakan Mamdani Fuzzy Inference System dengan komponen:

  • 7 variabel input dibagi menjadi 3 kategori (akademik, fisik, non-akademik), masing-masing dengan 3 himpunan fuzzy (rendah/sedang/tinggi atau sedikit/sedang/banyak).
  • 1 variabel output (tingkat stres) dengan 4 himpunan: rendah, sedang, tinggi, kritis.
  • 35 aturan IF-THEN dibagi 6 kategori: Akademik, Fisik, Non-Akademik, Compound (cross-domain), Positif (kompensasi), Baseline (single-variable coverage).
  • Operator agregasi: AND = MIN (T-norm), OR = MAX (S-norm), agregasi output = MAX clipping.
  • Defuzzifikasi centroid pada domain output [0, 100] dengan 200 sampel.
  • Forward chaining untuk menghasilkan rekomendasi intervensi kontekstual berdasarkan kombinasi input dan skor output.

Sumber Pengetahuan

Basis aturan dirumuskan dari kombinasi:

  • Literatur skala stres akademik standar (PSS-10 Perceived Stress Scale, Academic Stress Scale)
  • Wawancara terbatas dengan teman sejawat lintas fakultas
  • Konsultasi dengan literatur unit konseling mahasiswa (Gadjah Mada Medical Center)
  • Pengetahuan domain dari pengalaman langsung peneliti sebagai mahasiswa prodi dengan beban praktikum tinggi

Batasan

  • Sistem ini adalah alat self-awareness indikatif, bukan diagnosis klinis. Tidak dapat menggantikan psikolog atau konselor.
  • Input bersifat subjektif (self-report); akurasi hasil bergantung pada kejujuran pengguna.
  • Tidak terintegrasi dengan sistem akademik (Simaster, Elok); semua input dimasukkan manual.
  • Basis aturan dirancang berdasarkan konteks mahasiswa Indonesia, khususnya UGM; mungkin perlu adjustment untuk konteks lain.
  • Tidak menyimpan riwayat: setiap sesi independen.

Cara Penggunaan

  1. Buka halaman Diagnostik dan isi 7 variabel input sesuai kondisi Anda saat ini.
  2. Sistem secara real-time menampilkan skor stres, level, faktor dominan, dan rekomendasi.
  3. Telusuri Visualisasi untuk memahami posisi input Anda pada fungsi keanggotaan tiap himpunan fuzzy.
  4. Buka Rule Trace untuk melihat aturan-aturan yang ter-fire dan bagaimana skor terbentuk.
  5. Jika skor Anda di zona TINGGI atau KRITIS berkelanjutan, jadwalkan konsultasi dengan Gadjah Mada Medical Center (GMC) atau unit konseling fakultas.

Project Akhir

Sistem ini dibangun sebagai project akhir mata kuliah Kecerdasan Artifisial, S1 Elektronika dan Instrumentasi, FMIPA Universitas Gadjah Mada, semester genap 2025/2026.

Penafian. StresLens tidak menyimpan atau mengirim data Anda. Semua perhitungan dilakukan lokal di browser. Tidak ada server backend, tidak ada analytics, tidak ada tracking.
